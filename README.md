# Nolan Dela Rosa - Portfolio

This is my personal portfolio website, built to showcase my projects as a Full-Stack Developer.

🔗 Live Site: https://nolandelarosa.com

---

## Overview

The site is designed to be simple, clean, and easy to navigate.
It highlights a smaller number of meaningful projects with direct access to live demos and source code.

---

## Features

- Responsive, minimal UI
- Project showcase with GitHub + live demo links
- Resume access
- Custom domain deployment via Vercel

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- Vercel (deployment)

## Projects Featured
- **CareerPath** - Full-stack job tracking application
- **Pisayian Data Cleaner** - Backend data processing tool (Kapwa Codefest 2025)
